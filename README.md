# Revele-ai

## Comandos para acessar a base de produção:

Comando para fazer deploy para railway:
```
railway up 
```
Comando para criar um dominio:
```
railway domain 
```
Acesso a documentação pelo Swagger
```
https://revele-ai-api-production.up.railway.app/swagger-ui/index.html#/deputado/deputados
````
